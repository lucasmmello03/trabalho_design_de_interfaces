O que é Git?

Git é um sistema de controle de versão, criado por Linus Torvalds. Ele permite registrar alterações feitas em arquivos ao longo do tempo. Assim é possível voltar para versões anteriores do código ou acompanhar quem fez cada modificação em um projeto.

O que é GitHub?

GitHub é um serviço online que hospeda repositórios Git. Ele permite armazenar projetos, colaborar com outras pessoas e acessar códigos de qualquer lugar. Além disso, oferece ferramentas como issues, pull requests e documentação de projetos.

Comandos básicos do GitHub

  • git init - cria um repositório
  • git add - adiciona arquivos ao controle de versão
  • git commit - salva alterações no histórico
  • git push - envia o código para o GitHub
  • git pull - atualiza o projeto com alterações remotas
